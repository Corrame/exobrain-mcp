# 安装 Exobrain MCP Server

这份文档是一份给 AI 助手的自动化安装说明书（工作流）。
当你作为 Agent 被要求执行安装这里的 MCP 服务时，请遵循以下核心步骤，**切勿硬编码任何可能随设备变化的系统级配置文件的绝对路径**。

## 给 AI 助手的极简安装指令 (Agentic Workflow)

1. **解析本地运行环境**：
   利用终端工具解析并获取当前项目中虚拟环境里的 Python 解释器绝对路径（例如当前工作区下 `.\.venv\Scripts\python.exe` 的真实绝对路径）。

2. **解析服务入口点**：
   获取当前项目中 `exobrain-mcp/server.py` 的真实绝对路径。

3. **锚定目标配置文件（动态获取）**：
   不要去盲猜操作系统的全局路径（如 `%APPDATA%`），用户的 IDE 或 MCP 客户端的配置文件可能被高度定制或更改过位置。
   **请直接询问用户**：“你想把 MCP 注入到哪个客户端配置中？请提供配置文件的绝对路径（例如 `claude_desktop_config.json` 或 `mcp.json` 的路径）。如果不确定或希望仅在此项目生效，请回复`局部生效`，我将直接为你配置并写入工作区的 `.vscode` 配置中。”

4. **执行配置注入**：
   等待用户提供准确路径或给出指示后，读取该目标 JSON 配置文件。检查并确保存在 `mcpServers` 字典节点。
   在其中新建或更新 `personal-memory` 对象：将第一步获取的路径填入 `command`，将第二步获取的路径填入 `args` 数组。

5. **验证并收尾**：
   安全保存该配置文件。给用户输出一段友好的结果确认，并提醒用户重启相应的宿主客户端以使得新的节点生效。
